# Leaderboards

## Goal
Rank humans, agents, and model backends on the same scenario packs.

## Tracks
- Human teams.
- AI agents.
- Browser-native traffic models.
- SUMO-backed runs.
- Future SOTA model adapters.

## Scoring
- Scenario coverage.
- Delay prediction accuracy.
- Calibration.
- Schema validity.
- Reproducibility.
- Model-vs-real signal interval agreement once real intervals are available.
