# Papers

Store paper metadata, citations, methodology notes, model cards, and any downloaded open-access PDFs.

## Layout

- `manifest.json`: generated paper index for the website.
- `files/`: downloaded PDFs or other accessible paper assets.
- `README.md`: folder contract and download policy.

## Policy

- Keep the corpus Timisoara-first.
- Store metadata even when a paper is paywalled.
- Download PDFs only when the source is openly accessible or directly exposed.
